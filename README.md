# TheSkyLib
A tool with Python Scripts useful for owners of Skylanders™.

### About this fork:
I am creating this for preservation purposes, in case the main SkyLib disappears for some reason.

If [Zillion](https://github.com/DevZillion) updates the main repo, I will follow the changes.
